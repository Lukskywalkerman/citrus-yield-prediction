# citrus-yield-prediction
ML framework for the Brazilian citrus belt using CatBoost (R²: 0.925). Integrates 40 years of satellite indices (NDVI/LST) masked by MapBiomas, climate data, and global NPK fertilizer prices. Includes SHAP analysis, GEE scripts, and spatiotemporal GIFs for Bebedouro and Casa Branca. Data: 1985–2024.
